Adding a New Brush

  • Adding a New Brush

    • If we want to add the new brush to the FlashPhoto, we have to create a new brush tool .cpp and a new brush tool .h. The structure of this function that will be the same the structure of the Pen, Highlighter, Eraser and so on. We just change a little bit to convinience with the tool that we want to create it. For example, we have the code of the Pen tool like that

    #include "TPen.h"
    #include "MConstant.h"
    TPen::TPen()
    {
    m_mask = new MConstant(1.0, 1.0);
    }
    std::string TPen::getName()
    { return "Pen";
    }

    #ifndef TPEN_H
    #define TPEN_H
    #include "Tool.h"
    class TPen : public Tool
    {
    public:
    TPen();
    std::string getName() ;
    };
    #endif

    With Pen tool, we have to create the MConstant.cpp and MConstant.h to run for the Pen tool. With the new brush that we want to add to the paint program, we do the same like the Pen tool depanding on what specific tool we want to write about.

    After that, we will define the new brush tool in the ToolFactory.cpp. We will name the new brush tool in the Tool* ToolFactory::createTool(int toolID). That is some information to add the new brush to the project.
